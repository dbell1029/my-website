# david-bell.net

My personal website — built with Astro and Tailwind CSS.

## Stack

- [Astro](https://astro.build) — static site framework
- [Tailwind CSS](https://tailwindcss.com) — styling
- [Dante](https://github.com/JustGoodUI/dante-astro-theme) — base theme
- [EB Garamond](https://fonts.google.com/specimen/EB+Garamond) — serif font
- Deployed on [Netlify](https://netlify.com)

## Local Development

```bash
npm install
npm run dev
```

Site runs at `localhost:4321`.

## Structure

    src/
    ├── content/      # Blog posts and projects (Markdown)
    ├── data/         # Site config
    ├── pages/        # Page routes
    ├── styles/       # Global CSS and theme
    └── components/   # Astro components
